IFIX ROAD is an croud-sourcing based road status application. This application mainly support for Android and iOS based mobile applications.
Web application to monitor the road anomalies (e.g. Speed breakers, mud pits, grains, damaged Roads, road accidents and accident fire etc.).Police Department can easily updated with the latest road accidents and accident fire which were confirmed by more than five people using web application. Therefore it is going to be an easy task to clear the road or get some actions for those accidents or fire situations. And also Police Department can send road clear notification to the application.
